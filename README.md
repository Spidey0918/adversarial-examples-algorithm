每次提交一定要切换到自己的分支,在自己的分支里提交文件

* `git branch -a` (查看所有分支)\
* `git checkout lhc` (切换到自己名字下的分支)


提交文件操作：

* `git add README.md`\
* `git commit -m "this is a test" `(这里要写清楚你干了什么)\
* `git push` (最终提交)
